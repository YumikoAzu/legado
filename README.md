# 吾爱读书 
找不到一款完全满意的阅读APP。（完美主义。。。。）  
fork了大佬的legado代码，整理了一些不太习惯的地方。  
抽时间实现一下。  
好多年没写代码了。 

## 计划 
* 页眉页脚颜色变浅（实现）
* 时间和电量合并（实现）
* 更改默认封面样式
* 阅读界面沉浸式设置配色
* 阅读记录超过10分钟以上才显示
* 排版优化
* 修复部分epub书籍兼容性
